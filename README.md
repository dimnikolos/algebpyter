# 📐 Μαθηματικά Α΄ Γυμνασίου με Python

[![Άνοιξε στον browser](https://jupyterlite.rtfd.io/en/latest/_static/badge.svg)](https://dimnikolos.github.io/algepyter)

Διαδραστική παρουσίαση της ύλης **Μαθηματικών Α΄ Γυμνασίου** μέσω Jupyter Notebooks και Python.  
Οι μαθητές μπορούν να εκτελούν κώδικα Python κατευθείαν στον browser τους, χωρίς να χρειάζεται εγκατάσταση τίποτα.

## 🚀 Δοκίμασέ το τώρα

➡️ **https://dimnikolos.github.io/algebpyter**

Κάνε κλικ στο παραπάνω link, επέλεξε ένα notebook και πάτα **Run → Run All Cells** για να ξεκινήσεις!

## 🛠️ Τεχνολογία

- **[JupyterLite](https://jupyterlite.readthedocs.io/)** — Jupyter που τρέχει εξ ολοκλήρου στον browser (WebAssembly)
- **Python / Pyodide** — Εκτέλεση Python χωρίς server
- **GitHub Pages** — Δωρεάν hosting

## 📋 Απαιτήσεις

Μόνο ένας σύγχρονος browser:
- Firefox 90+
- Chrome / Chromium 89+
